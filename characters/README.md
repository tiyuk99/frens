# Character images

Put your own pixel character images here and add them in `script.js`.

1. Draw a 24×24 or 32×32 PNG (transparent background).
2. Save in this folder, e.g. `dino.png`, `robot.png`.
3. In `script.js`, add to the `CHARACTERS` array:
   ```js
   { id: "dino", label: "Dino", src: "characters/dino.png" }
   ```

They will appear as options on the start screen and on the progress bar.
